# Guia de Início Rápido

Este guia vai te ajudar a começar a usar o Wisedit rapidamente.

## Pré-requisitos

Antes de começar, certifique-se de ter:

- Node.js 18+ instalado
- Um editor de código (recomendamos VSCode)
- Conhecimento básico de terminal

## Instalação

```bash
npm install wisedit
```

## Configuração

Crie um arquivo de configuração:

```javascript
// wisedit.config.js
export default {
  // Suas configurações aqui
};
```

## Primeiro Uso

Aqui está um exemplo simples:

```javascript
import { Wisedit } from 'wisedit';

const editor = new Wisedit();
editor.init();
```
